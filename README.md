# Blood Donor Management and Prediction System

This project is developed using Python for managing blood donors and predicting donor eligibility using machine learning algorithms.

## Features
- Donor Eligibility Checking
- Blood Group Matching
- Logistic Regression Prediction
- Random Forest Prediction
- Confusion Matrix Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithms Used
- Rule-Based Algorithm
- Logistic Regression
- Random Forest Classifier

## Objective
To develop a smart system for blood donor management and prediction using machine learning techniques.
